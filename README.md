# 限界 GENKAI GM Dashboard

**Il tuo tavolo da Game Master, in digitale.**

![GENKAI GM Dashboard](https://maru.ga/img/hero.png)

---

## Cosa fa

GENKAI GM Dashboard è un'applicazione desktop pensata per i Game Master di giochi di ruolo da tavolo. Ti aiuta a preparare e gestire le sessioni con tutto a portata di mano, in un'unica interfaccia.

- **Documenti e handout** — Apri, visualizza e organizza appunti, mappe, schede PNG e file HTML direttamente nell'app
- **Calendario avventure** — Tieni traccia degli eventi di gioco con un calendario integrato
- **Telegram** — Invia messaggi, immagini e file ai tuoi giocatori direttamente dalla dashboard
- **Assistente AI** — Chiedi suggerimenti, genera descrizioni, nomi, incontri e idee al volo durante la sessione
- **8 temi visivi** — Scegli l'atmosfera giusta per la tua campagna
- **Avventure** — Gestisci più campagne in cartelle separate, ognuna con i suoi file e le sue impostazioni
